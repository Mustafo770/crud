# Блог API на Go (аналог Medium)

 REST API для блога.

### Функции
- CRUD для статей и комментариев
- Пагинация и поиск
- Лайки (доп. задание)
- Swagger документация: `/swagger/index.html`

### Технологии
- Go + Gin
- GORM + SQLite
- Swaggo для Swagger

### Как запустить
```bash
go mod tidy
go run main.go
